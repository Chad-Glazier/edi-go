# EDI

[![Go Reference](https://pkg.go.dev/badge/github.com/Chad-Glazier/edi.svg)](https://pkg.go.dev/github.com/Chad-Glazier/edi)

This repository holds the core EDI library.

EDI is a single-threaded, lightweight program to play [the Game of Amazons](https://en.wikipedia.org/wiki/Game_of_the_Amazons).
